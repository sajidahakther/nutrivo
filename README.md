## About Nutrivo
A health application that assists with improving nutrition knowledge and practices. Provides a simple nutrition guide to get you started. The ability to discover healthy recipes/meals that meet your dietary requirements. The option to monitor nutrient/calorie intake and view progress. Built with React Native, Redux, Firebase. 

## Build Instructions
- Requires Xcode to view the application on a simulator, install [here](https://developer.apple.com/xcode/).
- Download [zip file](https://gitlab.doc.gold.ac.uk/sakth050/nutrivo/-/archive/master/nutrivo-master.zip) or clone the repo by running `git clone https://gitlab.doc.gold.ac.uk/sakth050/nutrivo.git`
- Install dependencies: `npm install` 
- From the project root folder, open the **ios** folder then open **.xcodeproj** file on Xcode and run the app.
- The first build will take some time, but on a successful build it will launch the simulator.
