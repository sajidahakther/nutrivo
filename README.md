#### Build Instructions 

- Install dependencies: `npm install` in the root directory
- Start the react native packager, run `npm start` or `yarn run start` from the root directory
- Build the iOS app, by running `npm run ios` or `yarn run ios`. The first build will take some time, but it will start up a simulator.
