## About Nutrivo
A health application that assists with improving nutrition knowledge, attitudes and practices. Provides a simple nutrition guide to get you started;  a search tool to discover delicious recipes that meet your dietary requirements; a meal monitoring system to help you view your progress and reach your nutrition goals. Built with React Native, Redux, Firebase. 

## Build Instructions
- Requires Xcode to view the application on a simulator, install [here](https://developer.apple.com/xcode/).
- Download [zip file](https://gitlab.doc.gold.ac.uk/sakth050/nutrivo/-/archive/master/nutrivo-master.zip) or clone the repository to your local computer using the command line `git clone https://gitlab.doc.gold.ac.uk/sakth050/nutrivo.git`
- Install dependencies: `npm install` 
- From the project root folder, open the **ios** folder then open **.xcodeproj** file on Xcode and run the app.
- The first build will take some time, but on a successful build it will launch the simulator.
- If all has gone well, you will see an initial screen like the one above. :)
